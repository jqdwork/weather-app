
# Weather App

A weather dashboard built with React and Vite that displays current temperature, weather condition, and local time for major cities worldwide. Clicking a city reveals a historical temperature chart showing daily high and low temperatures.

## Getting Started

1. Clone the repository
```bash
   git clone https://github.com/jqdwork/Weather-App.git
   cd Weather-App
```

2. Install dependencies
```bash
   npm install
```

3. Start the development server
```bash
   npm run dev
```
